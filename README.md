# WordPress Site Health Toolkit

A lightweight WordPress diagnostics plugin designed to help troubleshoot common WordPress and WooCommerce support issues.

## Features

- PHP version checks
- WordPress version visibility
- WooCommerce detection
- REST API diagnostics
- WP_DEBUG checks
- Memory limit visibility
- WP Cron status

## Purpose

Built to simulate real-world troubleshooting workflows commonly used in technical customer support environments.

## Technologies Used

- WordPress
- PHP
- WooCommerce
- REST API
- CSS

## Future Improvements

- Plugin conflict detection
- Error log viewer
- WooCommerce checkout diagnostics
- Exportable support reports

## Real-World Use Cases

- Troubleshooting WooCommerce checkout issues
- Diagnosing WordPress plugin conflicts
- Reviewing REST API communication failures
- Monitoring login-related troubleshooting workflows
- Supporting WordPress technical issue resolution
